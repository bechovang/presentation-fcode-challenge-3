HTTP - Ngôn Ngữ Chung

HyperText Transfer Protocol

Ngôn ngữ giao tiếp giữa Client và Server

GET

Lấy dữ liệu

Xem tin tức

POST

Gửi dữ liệu mới

Đăng bài

PUT

Sửa dữ liệu

Đổi mật khẩu

DELETE

Xóa dữ liệu

Xóa bài viết

Tầm Quan Trọng Của Headers

Metadata - Thông tin phụ

Nhưng cực kỳ quan trọng!

# HTTP Headers Examples
Content-Type: application/json
User-Agent: Chrome/Windows
Authorization: Bearer xyz123
Accept-Language: vi-VN

Ví dụ thực tế

Content-Type: Báo server biết tôi gửi tiếng Anh hay tiếng Việt

User-Agent: Tôi đang dùng iPhone hay Laptop

Cookie/Token: Vé thông hành (chứng minh đã đăng nhập)

Stateless (Không Lưu Trạng Thái)

Chú cá vàng của Internet

Đặc điểm

Server "mất trí nhớ" sau mỗi request

Request thứ 2 không liên quan request thứ 1

Ý nghĩa

Muốn Server nhớ, Client phải gửi lại Token mỗi lần

Giống như: "Tôi là Tuấn, đã đăng nhập lúc sáng"

Server không có bộ nhớ dài hạn
